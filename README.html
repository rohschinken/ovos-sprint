ğŸ“š Features in Detail

Timeline View Modes

By Project Mode: View all team members assigned to each project. Expand projects to see individual member assignments.

By Team Member Mode: View all projects assigned to each team member. Expand members to see their project commitments.

Drag-to-Assign

Admins can click and drag across days to create assignments. The system will:

  • Warn about weekend/holiday assignments (if enabled)
  • Show visual feedback during dragging
  • Create assignments for all selected days
  • Color-code based on project status (confirmed/tentative)

Austrian Holidays

The system automatically detects and highlights all Austrian federal holidays:

  • Fixed holidays (New Year, Labour Day, National Day, Christmas, etc.)
  • Easter-dependent holidays (Easter Monday, Ascension Day, Whit Monday, Corpus Christi)
